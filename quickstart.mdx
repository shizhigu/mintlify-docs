## Install the CLI

<CodeGroup>
```bash npm
npm install -g mosscrm
```

```bash pnpm
pnpm add -g mosscrm
```

```bash yarn
yarn global add mosscrm
```
</CodeGroup>

Verify the installation:

```bash
moss --version
```

## Authenticate

Log in with your API key. You can create one from the [Dashboard](https://app.mosscrm.com) under **Settings > API Keys**.

```bash
moss auth login --key moss_k_your_api_key_here
```

Confirm you're authenticated:

```bash
moss auth whoami
```

## Create your first records

### Add a contact

```bash
moss contact create \
  --name "Jane Smith" \
  --email "jane@acme.com" \
  --phone "+1-555-0100" \
  --title "CTO"
```

### Add a deal

```bash
moss deal create \
  --title "Acme Corp - Enterprise" \
  --value 25000 \
  --stage lead \
  --contact-id <contact-id>
```

### Schedule a follow-up

```bash
moss followup create \
  --title "Send proposal to Jane" \
  --due-date 2026-02-15 \
  --contact-id <contact-id> \
  --priority high
```

### Check your pipeline

```bash
moss pipeline show
```

## Use JSON output

Every command supports `--json` for structured output, perfect for scripting and AI agents:

```bash
moss contact list --json | jq '.data[].name'
```
